Логирование обязательное:

-Старт бота
-Начало выполнения функции = имя функции
-Логирование каждой переменной в функции
-Начало выполнения цикла = имя ключевого элемента цикла
-Логирование каждой переменной в каждом цикле
-Окончание выполнения цикла = имя ключевого элемента цикла
-Окончание выполнения функции = имя функции
-У каждой функции должен быть "try: except Exception" и "Окончание выполнения функции = имя функции"
-У каждого цикла должен быть внутри except "try: Exception as e:" и "pass"

Все это вроде покрывается следующими конструкциями:

Блоки
---------------------------
	Функция:
		try
		   лог начала функции "имя функции"
		   ...
		   лог окончания функции "имя функции"
		except
		   ...
		   лог окончания функции "имя функции"
---------------------------
	Цикл:
	for\loop\while...
		try
			лог начала цикла "имя ключевого элемента цикла"
			...
			лог окончания цикла "имя ключевого элемента цикла"
		except
			...
			лог окончания цикла "имя ключевого элемента цикла"
---------------------------
Отправка уведомлений в tg:
    try
		bot.send_message...
	except telebot.apihelper.ApiTelegramException as e:
		...
		pass\continue в зависимости от наличия цикла выше
	except Exception as e:
	    ...
		pass\continue в зависимости от наличия цикла выше
---------------------------
Блок exception:
	Печать в консоль обязательна: шага, функции "имя функции", цикла "имя ключевого элемента цикла", текста ошибки
	Обязательна нотификация администраторов, что произошла ошибка: bot.send_message(346573500, ....) - #Даниловский
	Обязательно вписать еще одного инженера из L1
	Обязательна нотификация администраторов, что произошла ошибка: bot.send_message(218714164, ....) - #Полиров только для тестов
	Если это касается запроса пользователя, а не цикла в роботе, то обязательна нотификация пользователя, что произошла ошибка, извиниться, сказать, что уведомили администраторов
	
---------------------------
Переменные:
	При инициализации\изменении\присвоении должна быть обязательна печать в консоль: "имя функции", "имя ключевого элемента цикла", "имя переменной", "значение"

---------------------------
Курсоры:
	Отнестись как к переменной + cursor.rowcount
---------------------------
